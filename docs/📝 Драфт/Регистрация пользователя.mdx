# Регистрация пользователя

Авторизация <i class="fa-solid fa-file-user"></i> пользователя и его регистрация происходят в отдельном проекте, в котором пользователь может управлять своим аккаунтом. Такая система позволяет использовать один аккаунт во всех будущих проектах или серверах, вся информация об аккаунте находится в этом проекте.

При первом входе на проект пользователь может зарегистрировать новый аккаунт:

> Регистрация аккаунта через email и пароль, пользователю.
>
> - В первой стадии нужно ввести уникальный email, пароль, уникальный никнейм.
> - Во второй стадии нужно ввести код подтверждения с почты.
> - В третей стадии нужно загрузить аватар или оставить дефолтный, написать о себе.

> Регистрация аккаунт через <i class="fa-brands fa-telegram"></i> Telegram.
>
> - В первой стадии достаточно нажать одну кнопку авторизации через телеграмм и подтвердить действия.
> - Во второй стадии нужно ввести аватар (система попытается взять аватар <i class="fa-brands fa-telegram"></i> из телеграмма), написать о себе.

<CautionBlock>Аватар используется в не-игровых чатах и в некоторых системных частях, таких как системный форум, системные чаты.</CautionBlock>
